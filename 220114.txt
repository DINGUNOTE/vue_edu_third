# 데이터 호출 시점
1. 컴포넌트 라이프 사이클 훅
  - created : 컴포넌트(인스턴스)가 생성되자마자 호출되는 단계, 마운트 이전이므로 $el 속성을 아직 사용할 수 없다.
2. 라우터 네비게이션 가드
  - 특정 url에 접근하기 전에 동작들을 정의하는 속성(특정 url을 접근하지 못하게 막거나, 인증을 확인하는 등)
  - beforeEnter : to - 이동할 URL의 라우팅 정보, from - 현재 URL의 라우팅 정보, next - next가 호출 되어야 해당 URL로 이동이 가능하다.
* 시점 차이
  - 라우터 네비게이션 가드가 created보다 먼저 실행된다.

# Async & Await
* 자바스크립트 비동기 처리 패턴의 최신 문법, Promise와 Callback에서 주는 단점들을 해결하고 비동기적 사고 방식에서 벗어나 동기적(절차적)으로 코드를 작성할 수 있게 도와준다.
